# **App Name**: Legacy Lakeside Grill

## Core Features:

- Hero Section: Full-width hero section with headline 'Fine Dining. Great Company. Unrivaled Views.' and CTA buttons 'View Menu' and 'Book a Table.'
- Our Story: Section highlighting the restaurant's legacy, local ingredients, and lakeside atmosphere.
- Menu Modal: Interactive JavaScript modal for displaying menu categories (Signature Steaks, Fresh Catch, Craft Cocktails, Desserts) with premium pricing.
- Gallery Grid: Responsive 3-column grid showcasing food and lake-view photos.
- Contact & Location: Integrated Google Maps link and a functional 'tel' link for the phone number.
- Live Open Status: Script to check and display if the restaurant is currently open based on operating hours.

## Style Guidelines:

- Primary color: Deep Navy (#1B263B) for a sophisticated and calming feel.
- Accent color: Sunset Gold (#E09F3E) for highlights and call-to-action elements, providing warmth and elegance.
- Background color: Crisp White (#FFFFFF) for a clean and modern aesthetic, creating a striking contrast with the navy and gold.
- Headline font: 'Alegreya', a serif font, for headings, to provide an elegant and literary feel. Body font: 'PT Sans', a sans-serif font, for readability in body text.
- Subtle 'gold-glow' hover effect on all buttons.
- Mobile menu 'hamburger' to 'X' transformation on click.
- Full-width, atmospheric hero section with clear call-to-action buttons.
